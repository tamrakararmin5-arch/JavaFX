# BookMatch — Book Recommendation System

A JavaFX-based book recommendation system with MySQL database integration.

## How to run
1. Import the project into Eclipse
2. Make sure MySQL is running with the dbconnect database set up
3. Run the SQL setup script once to create tables and insert books
4. Update DB_PASS in Main.java if needed
5. Run Main.java as a Java Application

## Requirements
- Java 21+
- JavaFX SDK 21
- MySQL 8.0
- mysql-connector-j JAR in the build path
